## 🚀 What We Build

Back2Base creates **containerized AI code tooling** — developer tools that bring the power of modern AI into isolated, reproducible, and portable container environments. No messy local setups. No dependency drift. Just clean, consistent, AI-assisted development wherever you run it.

---

## 🛠️ Our Philosophy

- **Containerized by default** — every tool ships as a container, ensuring identical behaviour across local, CI, and production environments.
- **AI-first workflows** — built-in intelligence to help developers write, review, and reason about code faster.
- **Minimal footprint** — lean images, fast startup, zero bloat.
- **Open and hackable** — designed to fit into your existing pipelines, not replace them.

---

## 📦 Getting Started

Our tools are distributed as container images. Pull and run with a single command:

```bash
docker pull ghcr.io/back2base/<tool>:latest
docker run --rm -v "$(pwd)":/workspace ghcr.io/back2base/<tool>:latest
```

Check individual repository READMEs for tool-specific flags and configuration options.
